计算

映像 Image

映像（Image）是一个包含了软件及必要配置的机器模版。 作为基础软件，操作系统是必须的，您也可以根据自己的需求将任何应用软件 （比如，数据库、中间件等）放入映像中。

主机 Instance

主机（Instance）是以虚拟机器的形式运行在泰然云中的映像副本。 基于一个映像，您可以创建任意数量的主机。在创建主机时， 您需要指明 CPU 和内存的配置。泰然云允许您任意指定 CPU、内存的数量， 也允许您在主机创建之后随时再行调整。

网络

私有网络 Vxnet

私有网络 (Vxnets) 用于主机之间互联，它类似物理世界中使用交换机（L2 Switch）组成的局域网。私有网络与基础网络之间是100%隔离的。

路由器 Router

路由器用于多个受管私有网络之间互联，并提供多项附加服务，包括： DHCP 服务、端口转发、隧道服务、VPN 服务和过滤控制。

DHCP 服务

DHCP 服务为您的私有网络提供 IP 地址分配。分配的地址都是固定地址，即一个主机在其生命周期内获得的地址是保持不变的。

端口转发

添加端口转发规则，允许来自 Internet 或者 基础网络(vxnet-0) 对您的私有网络内部服务的访问。

隧道服务

隧道服务可以将多个不同地域的局域网连接在一起，形成兼顾公有私有的混合云计算环境，助您有效管理多地域的数据中心。

VPN 服务

VPN 服务使得您可以远程安全地拨入泰然云中的私有网络。

过滤控制

缺省情况下，路由器所管理的私有网络之间是可以相互连通的，你可以通过设置过滤规则来控制路由器内部私有网络之间的隔离。

公网IP EIP

公网 IP 地址是在互联网上合法的静态 IP 地址。 在泰然云系统中，公网 IP 地址与您的账户而非特定的资源（主机或路由器）关联， 您可以将申请到的公网 IP 地址分配到任意位于基础网络中的资源， 并随时可以解绑、再分配到其他资源，如此可以快速替换您的对外资源。

负载均衡器 Load Balancer

可以将来自多个公网地址的访问流量分发到多台主机上，并支持自动检测并隔离不可用的主机，从而提高业务的服务能力和可用性。同时，你还可以随时通过添加或删减主机来调整你的服务能力，而且这些操作不会影响业务的正常访问。负载均衡器还支持灵活配置多种转发策略，实现高级的自定义转发控制功能。

存储

硬盘 Volume

泰然云提供了两种规格的硬盘：容量型硬盘、性能型硬盘。

性能型硬盘

性能可达128MB/s，适用于对IO要求较高的应用，例如数据库等在线业务;

容量型硬盘

拥有更大的单块硬盘容积，最高可达5TB，适用于对容量要求较高的应用，例如文档存储等离线业务。

备份 Snapshot

备份（Snapshot）用于在块设备级别（block device level）上进行硬盘的备份与恢复， 可以同时对多张硬盘做备份（包括系统盘和数据盘），也可以对正在运行的主机做在线备份。 一张硬盘可以有多个备份链，每条备份链包括一个全量备份点以及多个增量备份点，您可以随时从任意一个备份点恢复数据。

在线备份

对运行中的主机做在线备份，无需停机重启，保证你的业务可用性

增量备份

一个全量备份点搭配多个增量备份点的方式，更节约存储空间

数据一致性

对同一主机的多块硬盘做并行备份，泰然云保证这些硬盘的数据一致性

数据库

关系型数据库 RDB

您仅需数十秒钟即可获得一个完整地基于 MySQL 的关系型数据库（RDB）服务，包括主从节点、自动备份、在线扩容以及监控告警等各种管理功能。泰然云数据库运行于私有网络内，结合泰然云提供的高性能硬盘和实时副本，您的数据安全将会得到最大限度的保护。

安全性

TRCloud关系型数据库运行于私有网络内，与其他租户100%二层隔离；严密账户保护，保障数据安全。

MySQL 存储引擎

基于性能和数据安全的考量，泰然云专门针对 InnoDB 存储引擎做了配置优化和调整， 未来也会对 InnoDB 进行更多更深入的增强。

主从同步

泰然云同时暴露了主从节点的 IP 以方便您的使用；多重实时副本保障数据存储介质安全，主从同步及备份确保数据不会丢失。

自动备份

默认情况下，数据库会每日备份，您的手工备份会连同每日备份一起最多保留12天。在数据库被彻底销毁前，您都可以选择从这些备份创建出全新的数据库，或者临时性维护节点。

缓存 Cache

缓存 (Cache) 可以为您提供高性能的缓存集群。一个集群包含多个缓存节点，支持主从及一主多从架构，确保高可用。另外，我们提供在线扩容，自动备份和监控告警等功能来帮助你更好的管理集群; 集群将运行于私有网络内，结合泰然云提供的高性能硬盘，在保障高性能的同时兼顾您的数据安全。

安全性

为了保障数据安全，缓存服务需要运行在受管的私有网络中。 所以在创建一个缓存服务之前，你至少需要拥有一个路由器和一个受管私有网络， 受管私有网络需要和路由器连接，并开启 DHCP 服务（默认开启）。

在线扩容

在缓存服务运行过程中，会出现服务能力不足或者容量不够的情况，你都可以通过扩容来解决。

手动、自动备份

缓存服务支持可持久化，你可以通过备份功能对缓存服务进行定期备份，对关键时间点的数据进行保护。 缓存服务支持自动备份，手动备份，基于备份创建新缓存等功能。 备份功能是块级别的 (block level)，最大程度保证数据安全， 同时备份过程不会影响正在运行的缓存服务。

全面监控及监控告警

我们提供了多达十几项的监控数据，来帮助用户随时了解缓存服务的运行状态和各项指标。 每一项监控都提供了历史监控和实时监控，最短时间间隔可以达到10s。其中可以为“内存使用率”和”被拒绝的 key 个数”创建监控告警策略。

安全

防火墙 Security Group

泰然云为每个用户提供了一个缺省防火墙。您也可以自建更多的防火墙。自建防火墙在初始状态下，所有端口都是封闭的，您需要建立规则以打开相应端口。

SSH 密钥 Keypair

为了提高安全性，保护您的主机，泰然云推荐您使用 SSH 密钥登录的方式。另外在创建 SSH 密钥时，支持用户输入公钥。
